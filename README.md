## Installation

    go get -u github.com/xyproto/alienpdf

## Usage

Generate a PDF file named `alien.pdf`:

    ./alienpdf

Which can then be opened, like this for instance:

    xdg-open alien.pdf

## Why?

This project is just for fun, and for trying out PDF generation from Go.
